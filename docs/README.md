This repository is part of a project-based AWS Cloud learning journey focused on:

* Understanding real cloud workflows
* Building practical hands-on projects
* Documenting architectural concepts
* Learning modern serverless systems

The goal is not only to build projects, but also to understand how production-style cloud systems are designed and implemented.
