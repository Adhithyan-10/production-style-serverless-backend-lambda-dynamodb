# Sample Data Folder

This folder contains sample JSON payloads and DynamoDB item structures used during testing and development.

These files help demonstrate how requests and data are structured in the serverless CRUD backend architecture.

---

# Included Files

## create-user-request.json

Sample request payload used for creating a new user.

## update-user-request.json

Sample request payload used for updating an existing user.

## sample-dynamodb-item.json

Example DynamoDB item structure stored in the database.

---

# Purpose of This Folder

This folder demonstrates:

* API request body structure
* JSON formatting for CRUD operations
* DynamoDB item representation
* Backend request testing workflow

---
