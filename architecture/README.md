## Architecture Overview

This project follows a serverless backend architecture using:

* API Gateway
* AWS Lambda
* Amazon DynamoDB
* CloudWatch
* SNS (for monitoring alerts)

The architecture demonstrates how modern cloud-native applications process requests without managing traditional backend servers.
